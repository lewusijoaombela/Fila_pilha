# Fila_pilha
Uso de comandos corretas para a fila e pilha
conforme ensinado na aula que a fila e a pilha são estruturas de dados para ordenação de posição de elementos na qual na fila o primeiro a entrar é o primeiro a sair diferente da pilha que é uma ordenação inversa.
os projetos se baseiam em sistema de numeração armezenam dados e mostram em posição conforme a sua estrutura utilizada.
